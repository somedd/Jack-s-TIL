  - 카드덱에 새로 추가한 메서드를 활용해서 카드스택 출력을 담당할 객체를 구현한다.
    - 클래스 이름, 변수 이름, 메서드 이름을 어떤 기준으로 작성했는지 readme 문서에 작성한다.
  - main에서 입력 기능을 잠시 제거하고, 바로 카드스택을 출력한다.
    - 일시적으로 기능을 제거할 때 어떤 방법이 좋은지 고민해본다.
    - 어떤 방식으로 기능을 제거했는지 주석을 남긴다.
